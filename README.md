# ZunSkills

陈明军的 AI 技能池，用来沉淀可复用的 Codex skills。

## Skills

- `resume-pdf-maker`：明尊简历PDF生成器。用于从简历内容或已有文件生成可确认、可预览、可导出的简历 PDF。

## Structure

```text
skills/
  resume-pdf-maker/
    SKILL.md
    agents/
    scripts/
    tests/
```

## Usage

把需要使用的 skill 复制或安装到本机 Codex skills 目录，例如：

```bash
cp -R skills/resume-pdf-maker ~/.codex/skills/resume-pdf-maker
```
